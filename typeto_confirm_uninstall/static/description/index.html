Type to Uninstall

Description

This Odoo module adds a safety mechanism to the uninstall process, requiring users to explicitly type the technical name of the module they intend to uninstall, thus preventing accidental removals.

How to Install

  • Add the module to your Odoo addons directory.
  • Restart Odoo.
  • Install

How to Use

Odoo will require user to type the module's technical name before confirming uninstallation.
